### 1. 여러 정수의 최대공약수, 최소공배수

### Solution

1. 우선 최대공약수를 구한다.

2. 전체 최대공약수(GCD) 를 구한다.

   A = aG, B = bG, C = cG, ..., G = GCD, a, b, ... 는서로소라고할때

   GCD(A, B, C) = GCD(GCD(A,B), C) 라는성질을이용한다.

3. 최소공배수(LCM)을 구한다.

   A = aG, B = bG, C = cG, … 라고할때 LCM = abcG = A/G * B/G * C/G * G 라는성질을 이용한다.
